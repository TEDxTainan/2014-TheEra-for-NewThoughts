2014TheEra-for-NewThoughts
==========================
## Required

- Install nodejs
- Install npm 

## Compile CSS file

First, you should install needed ruby gem (ruby version > 2.0.0)

    $ bundle install
    $ ./scss.sh

## Developer

第一次請先執行

    $ npm install
    $ bower install

安裝完之後打以下指令跑起 local web server

    $ npm start

然後在 browser 輸入 [http://localhost:8000](http://localhost:8000)
